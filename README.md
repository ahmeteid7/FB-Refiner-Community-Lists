# 🛡️ FB-Refiner Community Lists

نظام مشاركة قوائس الكلمات المتقدم لإضافة **FB-Refiner** - مركز التحكم الشخصي في Facebook

---

## 📖 نبذة عن المشروع

**FB-Refiner** هي إضافة متقدمة لتنظيف ملف تغذيتك على Facebook. هذا المستودع يوفر:

- 📤 **نظام رفع آمن** لقوائس الكلمات المخصصة
- 📥 **تحميل سهل** للملفات المشتركة من المستخدمين
- 👥 **مشاركة مجتمعية** - ساهم برائك وقوائسك الخاصة
- 🔒 **خصوصية كاملة** - جميع الملفات آمنة وشفافة

---

## 🎯 الميزات

✅ **موقع ويب سهل الاستخدام**
- واجهة عربية بالكامل
- تصميم حديث وسلس

✅ **رفع الملفات**
- اسحب وأفلت الملفات
- دعم ملفات `.txt` فقط
- حد أقصى 1MB لكل ملف

✅ **تحميل وتشغيل**
- تحميل الملفات مباشرة
- معاينة محتوى الملفات
- إحصائيات عن الملفات المتاحة

✅ **معالجة تلقائية**
- نظام `GitHub Actions` يعالج الطلبات تلقائياً
- حفظ آمن في المستودع

---

## 📋 متطلبات الملفات

### تنسيق الملف
```
كلمة1
كلمة2
كلمة3
...
```

**ملاحظات:**
- ملف واحد لكل سطر
- يجب أن يكون الملف بصيغة `.txt`
- الحد الأقصى: 1MB
- يمكن أن تكون الكلمات بأي لغة (عربي، إنجليزي، إلخ)

### مثال على ملف جيد:

```txt
كلمة سيئة 1
كلمة سيئة 2
slur1
offensiveword
عنصرية
```

## 🚀 كيفية الاستخدام

### 1️⃣ الرفع (Upload)

1. توجه إلى الموقع: https://ahmeteid7.github.io/FB-Refiner-Community-Lists/
2. اختر ملف `.txt` يحتوي على قائمة الكلمات
3. أدخل **اسم القائمة** (مثال: "Arabic Profanity Filter")
4. أدخل **الوصف** (اشرح ما تحتويه القائمة)
5. اضغط **رفع القائمة**

### 2️⃣ التحميل (Download)

1. افتح الموقع
2. اختر القائمة التي تريد من **القوائس المتاحة**
3. اضغط **⬇️ تحميل**
4. الملف سيُحفظ على جهازك تلقائياً

### 3️⃣ المعاينة (Preview)

1. اضغط **👁️ معاينة** بجانب أي ملف
2. ستظهر نافذة تعرض محتوى الملف

## ⚙️ كيفية عمل النظام

### المراحل:

1. **المستخدم يرفع ملف** 📤
   - يختار ملف `.txt` من جهازه
   - يكتب اسماً ووصفاً للقائمة

2. **النظام ينشئ Issue** 🔔
   - يُفتح `Issue` جديد تلقائياً
   - يحتوي على محتوى الملف

3. **البوت يعالج الطلب** 🤖
   - `GitHub Action` يتحقق من الملف
   - يضع الملف في مجلد `community-lists/`
   - يغلق الـ `Issue`

4. **الملف متاح للتحميل** ✅
   - يظهر على الموقع فوراً
   - يمكن تحميله في أي وقت
